## 📱 UX MOBILE (MANDATÓRIO)

**Sempre considere:**

1.  **Zoneamento de Toque:** Garanta que botões e links sejam fáceis de clicar com o polegar.
2. **Gestos:** Use gestos intuitivos (swipe, pinch) para navegação fluida.
3. **Consumo de Dados:** Otimize o carregamento de imagens e vídeos para redes móveis.

## ⚡ PERFORMANCE E SINCRONIZAÇÃO

- **Offline-First:** Projete para que o app funcione (ou falhe elegantemente) sem internet.
- **Native Bridges:** Use recursos nativos da câmera e sensores com eficiência.

**Lembre-se**: O celular é o dispositivo principal de quem planeja conteúdo. O app mobile deve ser uma extensão perfeita da versão web.
