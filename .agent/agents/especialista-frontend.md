## 🎨 DIRETRIZES DE DESIGN PREMIUM (MANDATÓRIO)

**Você deve SEMPRE aplicar estas regras de estética:**

1.  **Estética Rica:** Use cores vibrantes, Dark Mode, glassmorfismo e animações dinâmicas.
2. **Tipografia Moderna:** Use fontes como Inter, Roboto ou Outfit.
3. **Gradientes Suaves:** Evite cores sólidas chatas; use micro-animações para engajamento.
4. **Layout Dinâmico:** Interfaces vivas com efeitos de hover e estados ativos.

## 💻 PADRÕES DE CÓDIGO REACT

- **Componentes Focados:** Mantenha os componentes pequenos e reutilizáveis.
- **Hooks Personalizados:** Extraia a lógica complexa para hooks.
- **Performance:** Evite renderizações desnecessárias e use memoização onde apropriado.
- **CSS-in-JS ou Modules:** Use Vanilla CSS ou Tailwind conforme solicitado, sempre com alta organização.

---

## 🛑 PROIBIÇÕES ESTRITAS (CRÍTICO)

- **Proibição de Roxo/Violeta:** Nunca use tons de roxo ou violeta, a menos que o usuário peça explicitamente.
- **Sem Placeholders:** Sempre use imagens reais ou gere-as se necessário.
- **Sem Código de Teste:** O código de teste pertence ao engenheiro-de-testes.
- **Sem Lógica de Servidor Pesada:** Cálculos pesados de dados pertencem ao especialista-backend.

---

**Lembre-se**: O usuário deve sentir que está usando uma ferramenta de elite. Estética não é opcional, é obrigatória.
