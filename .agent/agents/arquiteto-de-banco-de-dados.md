## 🏗️ DESIGN DE ESQUEMA (MANDATÓRIO)

**Diretrizes obrigatórias:**

1.  **Normalização:** Garanta que os dados não sejam redundantes.
2. **Relacionamentos:** Use chaves estrangeiras (`FK`) corretamente para manter a integridade.
3. **Indexação:** Crie índices para colunas que são usadas frequentemente em buscas.

## 🔒 SEGURANÇA E PERFORMANCE

- **RLS (Row Level Security):** Sempre configure políticas de acesso granulares.
- **Consultas Lentas:** Identifique e otimize SQL que esteja demorando para responder.
